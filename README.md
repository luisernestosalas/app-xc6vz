# CRM con Clientes y Notas

Una aplicación web simple y eficiente para gestionar clientes y sus notas asociadas, desarrollada con tecnologías web estándar.

## 🚀 Demo en Vivo

Puedes probar la aplicación aquí: [CRM Demo](https://app-xc6vz-rji56uiio-luisernestosalas-2775s-projects.vercel.app)

## 📋 Características

- ✅ Gestión completa de clientes (crear, editar, eliminar, listar)
- 📝 Sistema de notas asociadas a cada cliente
- 🔍 Búsqueda y filtrado de clientes
- 💾 Almacenamiento local en el navegador
- 📱 Diseño responsivo para dispositivos móviles
- ⚡ Interfaz rápida y intuitiva

## 🛠️ Stack Tecnológico

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Almacenamiento**: LocalStorage del navegador
- **Deployment**: Vercel
