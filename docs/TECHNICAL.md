# Documentación Técnica - CRM con Clientes y Notas

## Información General

**Aplicación:** Sistema CRM (Customer Relationship Management)
**Funcionalidad principal:** Gestión de clientes y notas asociadas
**Stack tecnológico:** HTML + JavaScript (vanilla)
**Repositorio:** https://github.com/luisernestosalas/app-xc6vz
**URL de producción:** https://app-xc6vz-rji56uiio-luisernestosalas-2775s-projects.vercel.app

## Descripción del Sistema

Sistema de gestión de relaciones con clientes que permite:
- Almacenamiento y gestión de información de clientes
- Sistema de notas asociadas a cada cliente
- Interfaz web responsiva desarrollada en tecnologías frontend nativas

## Arquitectura Técnica

### Frontend
- **HTML5:** Estructura y marcado semántico
- **JavaScript (Vanilla):** Lógica de aplicación del lado cliente
- **CSS:** Estilos y presentación (implícito en stack web)

### Despliegue
- **Plataforma:** Vercel
- **Tipo:** Aplicación web estática/SPA (Single Page Application)

## Características Principales

1. **Gestión de Clientes**
   - Registro de información cliente
   - Visualización de datos de contacto
   - Búsqueda y filtrado

2. **Sistema de Notas**
   - Creación de notas por cliente
   - Historial de interacciones
   - Organización temporal
