# Manual de Usuario - CRM con Clientes y Notas

## ¿Qué es esta aplicación?

Es un sistema simple para gestionar clientes y sus notas. Te permite:
- Agregar clientes nuevos
- Guardar notas sobre cada cliente
- Ver toda la información de forma organizada

## Cómo acceder

**URL de la aplicación:** https://app-xc6vz-rji56uiio-luisernestosalas-2775s-projects.vercel.app

Simplemente abre esta dirección en tu navegador web.

## Cómo usar la aplicación

### 1. Agregar un cliente nuevo

1. Busca el formulario "Agregar Cliente"
2. Completa la información:
   - **Nombre**: Nombre completo del cliente
   - **Email**: Correo electrónico
   - **Teléfono**: Número de contacto
   - **Empresa** (opcional): Nombre de la empresa
3. Haz clic en "Agregar Cliente"
4. El cliente aparecerá en la lista

### 2. Ver la lista de clientes

- Todos los clientes se muestran en una lista
- Puedes ver: nombre, email, teléfono y empresa
- Cada cliente tiene botones para más acciones
